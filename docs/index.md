### Welcome to the DIY slow lamp wiki! 

The DIY slow lamp is a device that captures and displays the chromatic biography of your environment through an onboard camera that captures ambient, non-indexical color data. This ongoing record is visualized on the screen in the form of a spiral that grows larger with every new recording, representing the most ambient and vibrant colors of each hour. 

The device can be interacted with using a physical knob, which allows the user to scroll through the spiral to explore the data that has been captured to date. Additionally, the knob can be used to select a specific time-scale mode to display on the screen: the last 24 hours, the last 30 days, the last year, or all data, which is displayed as a continuous spiral.

{: .note }
Looking for a step-by-step build guide? Head directly over to the **[Tutorials](Tutorials/Tutorials.md)** page where you will find a streamlined, step-by-step guide designed to help you make your own DIY Slow Lamp.

### About the project

This project is an open-source, DIY branch of a long-term Research through Design (RtD) investigation taking place in [Homeware Lab](https://www.homewarelab.com). The goal of this project is to take the core values of "slow design" and extend them into a more accessible, participatory mode of making.

### How to use this wiki

This wiki serves as the central project repository for the DIY slow lamp. It contains everything you need to build your own version from scratch, modify its functions, and maintain or repair the device over its lifetime.
